# SIAH CO

Este proyecto esta enfocado para la eficiencia a la hora de almacenar datos de una ferreteria, la cual tiene como funciones adicionales
generar facturas, y alquilers.

## Índice

- [Requisitos](#Windows 64 bits, 6 RAM, Procesador I-5)
- [Instalación](#Node.js, Angular)
- [Configuración](#Servidor: Localhost:4000, Cliente: Localhost:4200, Variables GLOBALES: variables.env)
- [Estructura del Proyecto](#/

## Requisitos

Installar NODE.JS y Instalar Angular

## Instalación

Abrir la consola de comandos (cmd): cd (ruta-del-proyecto), cd (Angular) ,cd (Backend), cd (server), npm i
Abrir la consola de comandos (cmd): cd (ruta-del-proyecto), cd (Angular) ,cd (Frontend), cd (client), npm i

Para iniciar el servidor en la misma consola: cd (server), npm run dev
Para iniciar el cliente en la misma consola: cd (client), ng serve --o
